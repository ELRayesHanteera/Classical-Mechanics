Goldstein 3e — Chapter 1, Problem 11

Decide whether the force field $\mathbf{F} = yz\,\hat{\imath} + zx\,\hat{\jmath} + xy\,\hat{k}$ is conservative.

Book p. 31 (PDF p. 50) · Open with Live Server for auto-refresh

1. Physical setup

We are given a force field in 3D space with components:

$$F_x = yz, \qquad F_y = zx, \qquad F_z = xy$$

Core question: Can we find a scalar potential $V(x,y,z)$ such that $\mathbf{F} = -\nabla V$?

If yes, the field is conservative: work done moving a particle between two points depends only on endpoints, not the path. Equivalently, the work integral around any closed loop is zero.

2. Intuition

What does “conservative” mean physically? Imagine dragging a particle through this force field. If the field is conservative, you cannot extract net work by going in a circle — any energy you gain on one leg, you pay back on another. There is no “swirl” in the field.

Two equivalent tests (on a simply connected region like all of $\mathbb{R}^3$):

  1. Work integral: $\mathbf{F}\cdot d\mathbf{s}$ is an exact differential (path-independent).
  2. Curl test: $\nabla\times\mathbf{F} = \mathbf{0}$ everywhere.

Strategy: First use the work integral — it is the definition. Then cross-check with the curl.

Common trap: Checking only one component of curl, or integrating $yz\,dx$ with all coordinates held fixed at once. On a path integral, each leg holds different coordinates fixed.

3. Formal solution — Work integral

Step 1 — Line element and dot product

$$d\mathbf{s} = dx\,\hat{\imath} + dy\,\hat{\jmath} + dz\,\hat{k}$$

$$\mathbf{F}\cdot d\mathbf{s} = yz\,dx + zx\,dy + xy\,dz$$

Plain language: This is the tiny amount of work done along an infinitesimal displacement.

Step 2 — Is it an exact differential?

Try $V = -xyz$. Then:

$$d(xyz) = yz\,dx + xz\,dy + xy\,dz$$

So $\mathbf{F}\cdot d\mathbf{s} = d(xyz) = -dV$, and $\mathbf{F} = -\nabla V$. The integrand is exact.

Step 3 — Integrate along a path

From $(a,b,c)$ to $(x,y,z)$ via axis-aligned segments:

$$W = \int_a^x bc\,dx' + \int_b^y xc\,dy' + \int_c^z xy\,dz' = bc(x-a) + xc(y-b) + xy(z-c) = xyz - abc$$

Depends only on endpoints. Closed loop ($start = end$): $W = 0$.

3. Formal solution — Curl cross-check

$$(\nabla\times\mathbf{F})_x = \partial_y(xy) - \partial_z(zx) = x - x = 0$$

$$(\nabla\times\mathbf{F})_y = \partial_z(yz) - \partial_x(xy) = y - y = 0$$

$$(\nabla\times\mathbf{F})_z = \partial_x(zx) - \partial_y(yz) = z - z = 0$$

$$\boxed{\nabla\times\mathbf{F} = \mathbf{0}, \qquad V(x,y,z) = -xyz + C}$$

4. Sanity checks

CheckResult
Symmetry$F_x,F_y,F_z$ are cyclic permutations of $(x,y,z)$ — consistent with $V \propto xyz$
Origin$\mathbf{F}(0,0,0)=\mathbf{0}$
Cross-derivatives$\partial_x F_y = z = \partial_y F_x$, etc.

5. Takeaway

Yes — conservative. The work integral is exact with $V = -xyz$. Reusable habit: test $\mathbf{F}\cdot d\mathbf{s}$ first (definition), then curl as a quick cross-check.
